# Avito Monitor V2

## Как запустить бота
1. Создайте `.env` файл в директории `apps/avito-monitor-bot`. В `.env.template` есть пример заполнения этого файла.
   Необходимо указать токен телеграм бота, через которого будут происходить оповещения о результатах поиска.
   В этом же файле можно через переменную `CRON` настроить частоту запросов.
   При необходимости можно добавить переменную `HEADLESS=false`, чтобы видеть окно браузера.
3. Установите NodeJS 22+. Для удобства можно использовать NVM
4. Выполните `yarn install`
5. Выполните `yarn bot:dev` для запуска и отладки или `yarn bot:prod` для запуска в прод-режиме
